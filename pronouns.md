# Personal Pronouns in Greek

## Nominative  case

* εγώ (egó) — “I”
* εσύ (esí) — “you”
* αυτός (aftós) — “he”
* αυτή (aftí) — “she”
* αυτό (aftó) — “it”
* εμείς (emís) — “we”
* εσείς (esís) — “you”
* αυτοί (aftí) — “they” (masculine, mixed)
* αυτές (aftés) — “they” (feminine)
* αυτά (aftá) — “they” (neutral)

## Accusative case

* με (me) — “me”
* σε (se) — “you”
* τον (ton) — “him”
* τη(ν) (tin(n))  — “her”
* το (to) — “it”
* μας (mas)  — “us”
* σας (sas)  — “you”
* τους (tus) — “them”
* τις  or τες* (tis or tes) — “them”,  τις is used before a verb, τες after a verb.
* τα (ta) ) — “them”

__Example__: Σε βλέπω! — “I see you!”

## Genitive/Possessive case

* μου (mu) — “me”/“my”
* σου (su) — “you”/“your”
* του (tu) — “him”/“his” 
* της (tis) — “her”/“her”
* του (tu) — “it”/“its”
* μας (mas) — “us”/“our”
* σας (sas) — “you”/“your”
* τους (tus) — “them”/“their”
* τους (tus) — “them”/“their”
* τους (tus) — “them”/“their”

### Usage
* In possessive case pronoun follows the noun: Το βιβλίο μου — “My book”
* In genetive case pronoun preceeds the noun: Το μου αρέσει — “I like it”
